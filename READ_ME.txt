======================================================
		AI AGENT DISCORD Tutor
======================================================
AI Discord Tutor: an autonomous agent that proactively sends daily lessons, 
built from the student's own documents (revision) or the web (discovery), 
using MCP tools to plan, remember, format and deliver each lesson automatically

To make it work :
Add this files into your workspace of OpenClaw: 
-AGENTS.md
-HEARTBEAT.md
-SOULS.md
-TOOLS.md
-USER.md
-IDENTITY.md
And add MCP Tools from MCP Folder:
-TASK_MANAGER_MCP.py
-Memory_Logs_MCP.Py
-PDF_READER_MCP.py
-YOUTUBE_SCRAPPER_MCP.py
-LESSON_SENDER_MCP.py

Here you go !
